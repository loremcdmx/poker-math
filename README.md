# Poker Math

Небольшой учебный калькулятор по покерной математике с двумя рабочими режимами:

- `Быстрый калькулятор` для сайзинга, `breakeven FE`, `call odds`, `value : bluff`, `bluff share` и `MDF`;
- `Режим Игоря` с конвертером `value ↔ bluff`, режимом клиентского пота, блоком против коллбота и блоком `блеф с эквити`;
- интерактивными hero-чипами для быстрых переходов по экрану;
- переключением отображения ставки между процентами и дробями.

## Режимы

### Быстрый калькулятор

- ввод сайзинга через поле, слайдер и пресеты;
- короткие мнемоники по `risk / reward`;
- шпаргалка с популярными сайзингами.

### Режим Игоря

- конвертер из известного `value` в нужный `bluff` и обратно;
- `Режим клиента`, где банк вводится так, как он показан в руме;
- блок `Против коллбота` для рейзов;
- блок `Блеф с эквити` для semibluff-спотов;
- reference-таблица `банк 100` внизу страницы.

## Формулы

- `FE = Bet / (Pot + Bet)`
- `Call odds = (Pot + Bet) : Bet`
- `Value : Bluff = (Pot + Bet) : Bet`
- `Bluff share = Bet / (Pot + 2 * Bet)`
- `MDF = Pot / (Pot + Bet)`

Примеры:

- `1/2 pot -> 33.3% FE, 3:1 value:bluff`
- `1 pot -> 50% FE, 2:1 value:bluff`
- `2x pot -> 66.7% FE, 3:2 value:bluff`
